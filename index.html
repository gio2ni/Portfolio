🛡️ Application Anti-vol

Application Android de sécurité mobile qui surveille les mouvements du téléphone via l'accéléromètre. Dès qu'un mouvement suspect est détecté, une alarme sonore se déclenche au volume maximum. Seul un code PIN à 4 chiffres permet de l'arrêter.

ACTIVER LA PROTECTION

⚙️ Comment ça fonctionne

  1. Tu entres un code PIN à 4 chiffres et tu actives la protection.
  2. L'accéléromètre surveille en continu les mouvements du téléphone (seuil : 12.0 m/s²).
  3. Dès qu'un mouvement dépasse le seuil, l'alarme sonne au volume maximum.
  4. Seul ton code PIN peut couper l'alarme.

🔧 Difficultés & Solutions

  • Calibrer le seuil de sensibilité — trop bas, l'alarme se déclenchait pour un simple frôlement ; trop haut, elle ne se déclenchait plus.
    ✅ Solution : Dizaines de tests sur appareil réel → seuil fixé à 12 m/s², valeur qui distingue un vrai mouvement de la gravité naturelle.
  • Forcer le volume maximum malgré le mode silencieux.
    ✅ Solution : Utilisation de AudioManager.STREAM_ALARM, le seul canal audio Android qui ignore le mode silencieux.
  • Gérer le cycle de vie Android (onPause / onResume) — lors d'un appel entrant, le capteur s'arrêtait et la protection était perdue.
    ✅ Solution : Réenregistrement du listener dans onResume() pour reprendre la surveillance dès que l'app revient au premier plan.
  • Libérer le MediaPlayer sans crash à la fermeture.
    ✅ Solution : Appel de mediaPlayer.release() dans onDestroy() pour libérer proprement les ressources audio.

B3.3 Sécuriser les équipements et les usages des utilisateurs

L'application protège physiquement un smartphone contre le vol en déclenchant une alarme dès qu'un mouvement suspect est détecté. Seul un code PIN secret permet de la désactiver.

MainActivity.java — détection mouvement 
public void onSensorChanged(SensorEvent event) {
    float x = event.values[0];
    float y = event.values[1];
    float z = event.values[2];

    float magnitude = (float) Math.sqrt(x*x + y*y + z*z);
    float mouvement  = Math.abs(magnitude - SensorManager.GRAVITY_EARTH);

    // Si le mouvement dépasse le seuil → alarme
    if (mouvement > SEUIL && !alarmeActive) {
        runOnUiThread(() -> declencherAlarme());
    }
}

🔐 Générateur de mot de passe

Application web permettant de générer des mots de passe sécurisés et entièrement personnalisables. L'utilisateur choisit la longueur et les types de caractères. Tout se passe côté client : aucun mot de passe ne transite sur un serveur.

⚙️ Comment ça fonctionne

  1. Tu choisis la longueur du mot de passe (curseur).
  2. Tu sélectionnes les types de caractères : majuscules, minuscules, chiffres, symboles.
  3. Tu cliques sur Générer → un mot de passe aléatoire s'affiche instantanément.
  4. Tu copies en un clic dans le presse-papier.

💡 Point clé : tout se passe côté client, dans le navigateur. Aucune donnée ne transite sur un serveur — confidentialité totale.

🔧 Difficultés & Solutions

  • Garantir une vraie aléatoire — Math.random() seul ne garantit pas une distribution équitable entre les types de caractères.
    ✅ Solution : Construction d'un tableau de tous les caractères autorisés, puis pioche aléatoire dans ce tableau pour chaque position du mot de passe.
  • Interface responsive — faire fonctionner le design aussi bien sur mobile que sur ordinateur sans framework CSS.
    ✅ Solution : Utilisation de CSS Flexbox uniquement, sans dépendance externe, pour un design adaptatif léger.
  • Copie dans le presse-papier — navigator.clipboard.writeText() nécessite une connexion HTTPS pour fonctionner.
    ✅ Solution : Hébergement sur GitHub Pages qui fournit automatiquement le HTTPS, rendant la copie possible sans configuration supplémentaire.

script.js — génération du mot de passe 
function genererMotDePasse() {
    let caracteres = "";
    if (majuscules) caracteres += "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    if (minuscules) caracteres += "abcdefghijklmnopqrstuvwxyz";
    if (chiffres)   caracteres += "0123456789";
    if (symboles)   caracteres += "!@#$%^&*()_+";

    let mdp = "";
    // Pioche aléatoire dans le tableau de caractères
    for (let i = 0; i < longueur; i++) {
        mdp += caracteres[Math.floor(Math.random() * caracteres.length)];
    }
    return mdp;
}
